# ILoveCoffee - [NestJS Fundamentals](https://learn.nestjs.com/p/fundamentals) Course Project

## Description

[NestJS Fundamentals](https://learn.nestjs.com/p/fundamentals) course project repository

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
